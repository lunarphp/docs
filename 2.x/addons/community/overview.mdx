---

The Lunar community has built a growing collection of third-party packages that extend the platform's functionality. These packages are independently maintained by their authors and are not officially supported by Lunar.

<Info>
  Have an add-on to share? [Open a PR on the docs repository](https://github.com/lunarphp/docs) to add it to this page.
</Info>

## API

| Package | Description |
|---|---|
| [dystcz/lunar-api](https://github.com/dystcz/lunar-api) | A JSON:API-compliant API layer that connects the Lunar backend with SPAs, mobile apps, and other applications. Built on Laravel JSON:API. |

## Payments

| Package | Description |
|---|---|
| [webcraft/lunar-mollie](https://github.com/webcraft/lunar-mollie) | Mollie payment driver supporting credit cards, iDEAL, PayPal, Apple Pay, Bancontact, and more. Includes a Livewire checkout component. |
| [wychoong/lunarphp-mpgs](https://github.com/wychoong/lunarphp-mpgs) | Mastercard Payment Gateway Services (MPGS) Hosted Checkout integration with Livewire support. |

## Products

| Package | Description |
|---|---|
| [armezit/lunarphp-virtual-product](https://github.com/armezit/lunarphp-virtual-product) | Adds support for virtual and digital products such as subscriptions, memberships, and downloadable goods. |
| [armezit/lunarphp-purchase-limit](https://github.com/armezit/lunarphp-purchase-limit) | Allows defining purchase limitation rules based on quantity, amount, or both, scoped to products, variants, customers, or customer groups. |

## Disclaimer

Community add-ons are maintained by their respective authors. Lunar does not guarantee compatibility, security, or ongoing maintenance of these packages. Always review the source code and test thoroughly before using any third-party package in production.
